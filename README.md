# KFUPM EVENT HUB
  A platform to centralize KFUPM events 


## Setup and installation
  - Naviagte into "SWE363-Project" 
    - Inside command line: `cd .\SWE363-Project\`
  - Install dependencies
    - `npm install`
  - Run the server
    - `npm run dev`


## How to use
  - Start by choosing to login as student, organizer, or admin, or try singing in as a new user
  - Explore the page
  - Click on the avatar icon to log out
  - You can then try to log in as a different user access. i.e. student, organizer, or admin
